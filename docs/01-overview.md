# Overview

AnakKayu.id adalah website company profile dan CMS untuk brand mebel/interior kayu. Aplikasi menyediakan landing page modern classic, katalog produk, katalog layanan, portfolio, artikel, inquiry, CTA WhatsApp, share sosial media, SEO metadata, sitemap, robots.txt, dan modul social content import/generator.

Konsep visual publik mengikuti brief: header transparan di atas hero image, overlay gelap, headline besar, preview portfolio, about dua kolom, why choose us, layanan berlatar gold, gallery strip, cinematic banner, portfolio, dan footer gelap.
