# Public Website Workflow

Halaman publik:

- `/`: landing page.
- `/tentang-kami`: company profile.
- `/artikel`, `/artikel/{slug}`.
- `/produk`, `/produk/{slug}`.
- `/layanan`, `/layanan/{slug}`.
- `/portfolio`, `/portfolio/{slug}`.
- `/kontak` dan `/inquiry`.
- `/search?q=...`.
- `/sitemap.xml`.
- `/robots.txt`.

Data publik hanya mengambil item dengan status `published`.
