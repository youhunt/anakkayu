# Product, Service, Portfolio Workflow

Produk: `/admin/products`, publik `/produk`.

Layanan: `/admin/services`, publik `/layanan`.

Portfolio: `/admin/portfolio`, publik `/portfolio`.

Setiap detail publik memiliki share links dan CTA WhatsApp. Field harga dapat dibuat `by_request` untuk produk custom. Gallery disiapkan sebagai JSON agar bisa diperluas menjadi multi-upload/media library.
