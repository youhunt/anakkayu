# Skote Integration

Layout admin berada di `app/Views/layouts/admin.php`. Asset diarahkan ke:

- `public/assets/skote/css/bootstrap.min.css`
- `public/assets/skote/libs/jquery/jquery.min.js`
- `public/assets/skote/js/app.js`

Tempatkan paket Skote yang sudah disiapkan ke `public/assets/skote`. Fallback CSS custom tetap membuat dashboard dapat dibuka saat asset Skote belum disalin.
