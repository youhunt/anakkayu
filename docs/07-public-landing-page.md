# Public Landing Page

Homepage berada di `app/Views/public/home.php` dan CSS di `public/assets/anakkayu/css/site.css`.

Struktur landing:

1. Header transparan.
2. Hero image full width dengan overlay gelap.
3. Headline besar modern classic.
4. Floating portfolio preview cards.
5. About dua kolom.
6. Why choose us dengan gambar tengah.
7. Services section berlatar gold.
8. Horizontal gallery strip.
9. Cinematic banner.
10. Portfolio section.
11. Footer gelap.
